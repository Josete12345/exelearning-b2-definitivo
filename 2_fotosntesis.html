2- Fotosíntesis

La fotosíntesis

La fotosíntesis se podría definir como el proceso por el cual las plantas fabrican su propio alimento.

Para realizar la fotosíntesis las plantas necesitan: luz solar, dióxido de carbono, agua y clorofila.

Podéis ver aquí como se realiza el proceso de fotosíntesis:

Aquí puedes encontrar un vídeo para saber más acerca del tema:


Gracias a la luz, las plantas capturan dióxido de carbono (CO2) y expulsan oxígeno durante el día.

Por las noches sucede lo contrario, capturan oxígeno y liberan dióxido de carbono.

Este proceso facilita la vida en la Tierra, ya que nos proporciona el oxígeno que necesitamos para respirar.

Ahora que ya entiendes un poco mejor el proceso de Fotosíntesis, realiza las actividades dentro del apartado tareas.
